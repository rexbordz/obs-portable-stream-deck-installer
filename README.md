# OBS Portable Stream Deck Installer

One annoying thing that happens when you update your portable OBS is the Stream Deck plugin. For some reason, you need to recopy the Stream Deck OBS plugin files to your portable OBS install each time you update OBS, or else your Stream Deck won't connect to OBS. So every time you update your OBS, you need to visit Elgato's [tutorial page](https://help.elgato.com/hc/en-us/articles/15525011385229-Elgato-Stream-Deck-How-to-use-with-OBS-Studio-Portable-Mode) on how to install the Stream Deck plugin to a portable OBS to know which files to copy. This program eliminates that need. All you need to do is launch the included [exe](https://github.com/rexbordz/obs-portable-stream-deck-installer/releases/tag/v1.0.0) file, select your OBS portable installation path and click the install button!

## Quick Start
 - Make sure you have the [Stream Deck OBS plugin](https://elgato.com/download) installed
 - Launch [**ObsPortableStreamDeckInstaller.exe**](https://github.com/rexbordz/obs-portable-stream-deck-installer/releases/tag/v1.0.0)
 - Click **Browse** -> select your OBS portable installation folder
 - Click **Install / Update**
